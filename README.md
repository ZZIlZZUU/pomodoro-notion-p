# Minimalist Pomodoro Widget for Notion

A minimalist, aesthetic, and automated Pomodoro timer designed specifically to be embedded into Notion using GitHub Pages.

## Features
* **Continuous Loop:** Automatically transitions between Focus and Break sessions without manual intervention.
* **Sleek Animation:** Features a subtle, rotating conic-gradient border acting as a visual anchor.
* **Native Theme Matching:** Automatically adapts to Light Mode and Dark Mode based on your system or browser preferences.
